# Todo App - React Express PostgreSql Docker

## Install:
```
[Docker](https://www.docker.com/)   
```
## Usage
FIRST RUN
```
$ sudo docker-compose up
```
START
```
$ docker-compose up
```
STOP
```
$ docker-compose down
```
